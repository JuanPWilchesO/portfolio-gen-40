Principalmente tengo habilidades para el desarrollo web, haciendo uso de las tecnologías de HTML5, JavaScrip y CSS, adicionalmente conozco el lenguaje de c++ junto con la librería de Qt, cuento adicionalemte con conocimientos en analisis de datos haciendo uso de Python y sus principales herramientas como Pandas.

HTML

CSS

Estilado de paginas web haciendo uso de la tecnología CSS y sus diversas herramientas de manera creativa y priorizando la comodidad de los usuarios en el momento de interactuar con los sitios web que se construyen.

JavaScript

Programación de caracteristicas y diversos procesos asociados a paginas web. Implementacion de diversos componentes y funcionalidades para dar características únicas a los sitios web. C++

Manejo de la programacion básica y orientada a objetos en C++ para la construccion de diversas herramientas tecnologicas que se despliegan principalmente en CLI. Manejo del framework Qt para la construccion de aplicaciones de escritorio con interfaz amigable para los usuarios.
